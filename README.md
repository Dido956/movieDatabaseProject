## SoftUni Spring Advanced Project

**Movie Database Website**

This is a simple movie database website created for a university web project. It allows users to browse, search, and
view information about movies. The website is built using the following technologies:

* Front-end: HTML, CSS, Thymeleaf, Bootstrap 5
* Back-end: Java, Spring
* Database: MySQL

### Features

* Browse movies
* Search for movies by title, director, and cast
* View detailed information about movies, including trailers, reviews, and ratings
* Add movies to a watchlist
* Admin panel

To browse movies, click on the Movies link in the navigation bar and chose one of the options from the dropdown.

To search for a movie, enter the title, director, or cast member in the search bar and click the "Search" button.

To view detailed information about a movie, click on the movie poster.

To add a movie to your watchlist, click on the "Add to Watchlist" button.
